Final Project: Paper Prototype

The user will open up the website in the home page, and the design is meant for mobile. This home page places all the components for adding a new item to the schedule list. The date is placed on the right, while the container for the time, note, and image is slightly bigger to provide emphasis and grab user attention. The padding in-between helps make it less overwhelming for the eye to scan, and it is touch-screen compatible for the user to scroll down. Tapping on the +Add brings a new page overlay that the user can customize how the schedule item looks. It includes a small graphic representative icon for different things (optional), a time picker, a calender, and a text box. Tapping on the name (user sign-in) brings down a menu that allows the user to pick a timezone and turning off or on different features of the website (graphics, reminders, notifications, autodelete), as well as a color pallet choice for the interface if the user wants to personalize the site's appearance.
